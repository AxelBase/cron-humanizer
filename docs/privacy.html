Privacy Policy

Privacy Policy

Effective: 12 November 2025 • AxelBase

1. Zero Data Collection

AxelBase does not collect, store, or process any user data. Every tool — including the Cron Humanizer — runs entirely in your browser. Your cron expressions, inputs, and outputs never leave your device.

2. What We Do Not Track

  • No IP logging — We do not record your IP address.
  • No analytics — No Google Analytics, Plausible, or tracking pixels.
  • No cookies — No session, preference, or functional cookies.
  • No localStorage persistence — Input is cleared on reload.
  • No server logs — Hosted on GitHub Pages with no backend.

3. How Our Tools Work

All processing happens client-side using JavaScript:

  • Cron parsing via cron-parser
  • Humanization with custom logic
  • Next-run calculation using browser time APIs
  • Copy/share features use native browser functions

Your data is processed locally and discarded immediately.

4. Third-Party Dependencies

We use minimal, privacy-safe CDNs:

  • Bootstrap 5 CSS (jsDelivr)
  • date-fns (jsDelivr)

These load static assets only. No tracking scripts are included.

5. Security & Responsibility

Since no data is transmitted, data breaches are impossible. You control your environment. We recommend:

  • Using HTTPS (enforced)
  • Keeping your browser updated
  • Avoiding public networks for sensitive inputs

6. Compliance

By collecting nothing, we comply with GDPR, CCPA, POPIA, and all global privacy laws by default.

7. Changes

Updates will appear here with the new effective date.

Your privacy is not a feature — it’s the foundation.
Use AxelBase with complete confidence.
